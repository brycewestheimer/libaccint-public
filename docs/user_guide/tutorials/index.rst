Tutorials
=========

These tutorials guide you through increasingly complex uses of LibAccInt,
from basic integral computation to advanced performance optimization.

Tutorial Overview
-----------------

**Beginner**

1. :doc:`01-basics` - Core concepts: shells, basis sets, and engines
2. :doc:`02-one-electron` - Computing overlap, kinetic, and nuclear integrals

**Intermediate**

3. :doc:`03-two-electron` - Two-electron integrals and ERI computation
4. :doc:`04-fock-matrix` - Building Fock matrices with FockBuilder

**Advanced**

5. :doc:`05-gpu-acceleration` - Using the CUDA backend
6. :doc:`06-python-bindings` - Python API and NumPy integration

Prerequisites
-------------

Before starting these tutorials, you should:

- Have LibAccInt :doc:`installed <../installation>`
- Be familiar with basic quantum chemistry concepts
- Know C++ basics (for C++ tutorials) or Python (for Python tutorials)
