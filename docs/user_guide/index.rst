What is LibAccInt?
------------------

LibAccInt (**Lib**\rary for **Acc**\elerated **Int**\egrals) is a high-performance
molecular integral library for computational quantum chemistry. It provides:

- **GPU Acceleration**: CUDA support for modern NVIDIA GPUs
- **High Performance**: Optimized CPU paths with OpenMP and SIMD
- **Modern C++20**: Clean API using modern C++ features
- **Python Bindings**: NumPy-integrated Python interface
